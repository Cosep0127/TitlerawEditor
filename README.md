# /titleraw 可视化编辑器

Minecraft 基岩版 `/titleraw` 命令的可视化编辑器，Apple 极简风格。

## 功能

- **可视化组件编辑**：添加、删除、排序组件，支持 `text` / `selector` / `score` / `translate` 四种类型
- **格式化代码表**：29 种基岩版颜色代码 + `§l` `§o` `§k` `§r` 格式，点击复制 § 编码
- **命令导入**：粘贴完整命令（`/` 可不加）或纯 JSON，自动解析为组件
- **实时预览**：
  - 命令预览（格式化/压缩切换）
  - 视觉预览（支持 § 颜色/格式代码和 `\n` 换行渲染，使用 GNU Unifont 字体）
- **一键复制**：复制生成好的命令到剪贴板
- **操作保护**：清空组件和导入命令时有二次确认

## 使用

直接用浏览器打开 `index.html`，无需任何依赖。

## 文件

```text
├── index.html    页面骨架
├── style.css     Apple 极简样式
└── script.js     交互逻辑
```

## 界面

1. **目标 + 显示位置** — 目标玩家（支持 `@a` `@p` `@r` `@e[type=player]` 快捷按钮）和 title/subtitle/actionbar 类型选择
2. **命令导入** — 粘贴现有命令自动解析（折叠卡片，点击展开）
3. **格式化代码表** — 29 色 + 4 种格式，点击编码复制（折叠卡片，点击展开）
4. **组件列表** — 可视化编辑 rawtext 组件，支持排序和清空
5. **预览** — 使用 GNU Unifont 字体的标题效果模拟
6. **命令** — 最终命令展示，支持格式化/压缩切换和一键复制

## 颜色

基岩版支持 29 种颜色代码，在文本中使用 `§` + 代码字符即可应用：

- 16 种标准色：`§0`–`§f`
- 13 种材质色：`§g` minecoin_gold、`§h` 石英、`§i` 铁锭、`§j` 下界合金、`§m` 红石、`§n` 铜锭、`§p` 金锭、`§q` 绿宝石、`§s` 钻石、`§t` 青金石、`§u` 紫水晶、`§v` 树脂、`§w` 组队蓝

## 格式

| 编码 | 效果 | 说明 |
|------|------|------|
| `§l` | **粗体** | 加粗显示 |
| `§o` | *斜体* | 倾斜显示 |
| `§k` | 随机 | 字符随机闪烁 |
| `§r` | 重置 | 重置所有格式 |

## 注意事项

- 生成的命令格式为 `/titleraw <player> <type> {"rawtext":[...]}`
- 文本组件支持 `§` 颜色/格式代码和 `\n` 换行，渲染时自动解析
- selector 组件在预览中显示为 `§7[target]`，score 组件显示为 `§7[0]`
