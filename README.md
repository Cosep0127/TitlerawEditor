# /titleraw 可视化编辑器

Minecraft 基岩版 `/titleraw` 命令的可视化编辑器，Apple 极简风格。

## 功能

- **可视化组件编辑**：添加、删除、排序组件，支持 `text` / `selector` / `score` / `translate` 四种类型
- **格式化代码表**（弹窗）：29 种基岩版颜色代码，分标准颜色 / 材料颜色 / 其他三区展示，`§l` `§o` `§k` `§r` 格式，点击卡片复制 § 编码
- **命令导入**（弹窗）：粘贴完整命令（`/` 可不加）或纯 JSON，自动解析为组件
- **撤销/重做**：支持 Ctrl+Z / Ctrl+Y，最多保留 100 步操作历史
- **深色模式**：跟随系统偏好 + 手动切换，自动保存主题选择
- **状态持久化**：编辑内容自动保存到 localStorage，刷新页面不丢失
- **实时预览**：
  - 命令预览（格式化/压缩切换）
  - 视觉预览（支持 § 颜色/格式代码和 `\n` 换行渲染，使用 GNU Unifont 字体）
- **一键复制**：复制生成好的命令到剪贴板
- **操作保护**：清空组件和导入命令时有二次确认

## 使用

直接用浏览器打开 `index.html`，无需任何依赖。

## 界面

1. **目标 + 显示位置** — 目标玩家（支持 `@a` `@p` `@r` `@e[type=player]` 快捷按钮）和 title/subtitle/actionbar 类型选择（带滑动指示器动画）
2. **操作按钮** — 「导入指令」和「格式化代码表」按钮，点击弹出对应弹窗
3. **组件列表** — 可视化编辑 rawtext 组件，支持排序、撤销和清空
4. **预览** — 使用 GNU Unifont 字体的标题效果模拟（深色舞台背景）
5. **命令** — 最终命令展示，格式化/压缩切换（带滑动指示器）和一键复制

## 颜色

基岩版支持 29 种颜色代码，在文本中使用 `§` + 代码字符即可应用：

- **标准颜色**（16 种）：`§0`–`§f`
- **材料颜色**（12 种）：`§h` 石英、`§i` 铁锭、`§j` 下界合金、`§m` 红石、`§n` 铜锭、`§p` 金锭、`§q` 绿宝石、`§s` 钻石、`§t` 青金石、`§u` 紫水晶、`§v` 树脂
- **其他**（2 种）：`§g` Minecoin金、`§w` 组队蓝

## 格式

| 编码 | 效果 | 说明 |
|------|------|------|
| `§l` | **粗体** | 加粗显示 |
| `§o` | *斜体* | 倾斜显示 |
| `§k` | 随机 | 字符随机闪烁 |
| `§r` | 重置 | 重置所有格式 |

## 注意事项

- 生成的命令格式为 `/titleraw <player> <type> {"rawtext":[...]}`
- 文本组件支持 `§` 颜色/格式代码和 `\n` 换行，渲染时自动解析
- selector 组件在预览中显示为 `§7[target]`，score 组件显示为 `§7[0]`
